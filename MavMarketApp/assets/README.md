# MavMarketApp `assets/`

Static app branding assets referenced by Expo config.

## Files

- `icon.png` -> primary app icon
- `splash-icon.png` -> splash image
- `favicon.png` -> web favicon for Expo web target
- `android-icon-background.png`
- `android-icon-foreground.png`
- `android-icon-monochrome.png`

## Config usage

`app.json` references these files for iOS/Android/web icon and splash settings.

If you replace assets, update `app.json` paths and verify app launch/icon rendering on each platform.
